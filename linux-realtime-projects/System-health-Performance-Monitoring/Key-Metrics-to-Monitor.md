# Key Metrics to Monitor

* **CPU Utilization** - High CPU Utilization indicates bottlenecks in the system.
* **Memory Usage** - Amount of memory currently in usage.
* **DISK I/O** - Amount of data that is being read or written to the disk.
* **Network Traffic** - Amount of data being transmitted over the network.
* **Application Performance Metrics** - Metrics specific to the applications that running on the system like response time and throughput.
